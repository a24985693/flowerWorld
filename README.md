# 花花世界
模擬電商網站

## 專案說明
花花世界是一個模擬的電商網站，讓使用者可以瀏覽商品、加入購物車、購買商品等。
管理者可以在後台新增或修改商品、消息、優惠券等。


## DEMO
https://a24985693.github.io/flowerWorld/#/

## 網站功能

- **前台**
  - 商品介紹、商品分類
  - 最新消息
  - 收藏
  - 搜尋訂單
  - 購物車（調整數量、刪除單項、清空購物車及套用優惠券）
  - 結帳

- **後台**
  - 管理者登入 / 登出、驗證登入狀態
  - 管理商品、訂單、消息和優惠券（新增、編輯、刪除、上架及下架）


## 專案指令

### 安裝相關專案
```
npm install
```

### 運行環境
```
npm run serve
```

### 建構專案
```
npm run build
```
